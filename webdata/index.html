Welcome to the Main Site

This is a multi-site Nginx deployment demonstration. The site is served through Docker containers with SSL/TLS encryption and supports multiple paths.

Features

  • Docker containerization with Nginx
  • HTTPS with self-signed certificates
  • Multi-site configuration (/, /reloj, /admin)
  • SFTP file transfer support
  • HTTP to HTTPS redirect (301)
  • Basic authentication for admin area
